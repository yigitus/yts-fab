== Description ==

A Basic floating action button plugin for your website.

== Installation ==

1. Put all files into a folder and upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR 

1. Get the plugin from wordpress plugins menu
2. Activate the plugin through the 'Plugins' menu in WordPress
